冒聼聰麓 Critical / High Priority

    File uploads, deserialization, template injection (SSTI)

    JWT flaws, default creds, broken session tokens

  • Server-Side Request Forgery (SSRF)

  • Cross-Site Scripting (XSS)
    Reflected, stored, DOM-based. Test all input fields

    Horizontal/vertical privilege escalation. Access admin endpoints

    XML upload / parsing endpoints. Try DOCTYPE injection
冒聼聼隆 Medium / Low Priority
  • CSRF (Cross-Site Request Forgery)
    State-changing actions missing CSRF tokens
  • Security Misconfigurations
    Debug mode on, directory listing, exposed .env/.git
  • Sensitive Data Exposure
    API keys in JS, tokens in URLs, unencrypted PII
  • Subdomain Takeover
    Dangling DNS records pointing to unclaimed services
  • Open Redirect
    Redirect params that can point to external URLs
  • Information Disclosure
    Version numbers, stack traces, internal paths in errors
  • Clickjacking
    Missing X-Frame-Options / CSP frame-ancestors

    HSTS, CSP, X-Content-Type-Options absent
